Getting Started

Welcome to Leaf Singularity. This section will guide you through the basics of installation, setup, and creating your first project.

For video tutorials, visit our Community Forum where users share workflow guides and tips.

System Requirements

Component Minimum Recommended
Processor Intel Core i5 / AMD Ryzen 5 Intel Core i7 / AMD Ryzen 7
RAM 8 GB 16 GB or more
Storage 2 GB SSD 5 GB SSD
GPU Integrated Graphics NVIDIA GTX 1060 / RTX 2060 or equivalent
OS Windows 10 / macOS 10.14 / Ubuntu 18.04 Windows 11 / macOS 12+ / Ubuntu 20.04+
Internet 2 Mbps (for cloud sync) 10 Mbps (recommended)

Installation

Windows Installation

  1. Download the Windows installer (.exe) from Downloads
  2. Run the installer with administrator privileges
  3. Accept the license agreement
  4. Choose installation location (default: C:\Program Files\Leaf)
  5. Select additional tasks (desktop shortcut, start menu)
  6. Click Install and wait for completion
  7. Launch Leaf Singularity from Start Menu or desktop shortcut

macOS Installation

  1. Download the macOS installer (.dmg) from Downloads
  2. Double-click the .dmg file to mount it
  3. Drag the Leaf Singularity icon to the Applications folder
  4. Open Applications folder and double-click Leaf Singularity
  5. If prompted, approve security permissions in System Preferences
  6. Grant microphone and camera access when requested

Linux Installation

Ubuntu/Debian-based systems:

sudo apt update sudo apt install leaf-singularity

Or using AppImage:

chmod +x Leaf-Singularity.AppImage ./Leaf-Singularity.AppImage

Fedora/RHEL-based systems:

sudo dnf install leaf-singularity

Create Your First Project

  1. Launch Leaf Singularity
  2. Click "New Project" or use File → New
  3. Choose project type: Design, 3D Model, or Animation
  4. Set canvas size (or use presets: 1920x1080, A4, etc.)
  5. Click "Create" to initialize your project
  6. Your project is auto-saved to cloud every 15 minutes

Projects are stored locally and synced to cloud if you're logged in. You can access your projects from any device.

Feature Guides

Design Editor

Create and edit 2D designs with vector, raster, and text tools. The design editor includes:

  • Vector drawing and shape tools
  • Non-destructive layer effects and filters
  • Advanced typography with variable fonts
  • Artboards for multi-page designs
  • Component system for reusable elements

Real-Time Collaboration

Work with team members simultaneously on the same project:

  • Share project link with team members
  • See live cursor positions and edits
  • Leave comments and annotations on layers
  • Version history with rollback capability
  • Permission levels: view, edit, or admin

3D Modeling

Import and create 3D models directly:

  • Import OBJ, FBX, BLEND, GLB formats
  • Built-in 3D primitives and modeling tools
  • Material and texture editor
  • Real-time rendering with multiple light types
  • Export as models or rendered images/video

Motion Graphics & Animation

Create animated content with keyframe animation:

  • Timeline-based keyframe animation
  • Easing curves and animation presets
  • Motion blur and particle effects
  • Export as MP4, WebM, or GIF
  • Frame-by-frame export for maximum control

API Reference

Integrate Leaf Singularity with your applications using our REST API.

Authentication

All API requests require authentication using an API key:

Authorization: Bearer YOUR_API_KEY Content-Type: application/json

Get your API key from Account Settings → Developer → API Keys.

Base URL

https://api.leafsingularity.com/v1

Projects Endpoints

Method Endpoint Description
GET /projects List all user projects
POST /projects Create new project
GET /projects/:id Get project details
PUT /projects/:id Update project metadata
DELETE /projects/:id Delete project
POST /projects/:id/export Export project to PNG/SVG/MP4

Example Request

curl -X GET https://api.leafsingularity.com/v1/projects \ -H "Authorization: Bearer YOUR_API_KEY" \ -H "Content-Type: application/json"

Troubleshooting

Application crashes on startup

Solution: Clear the application cache and restart:

# Windows del %APPDATA%\Leaf\cache # macOS rm -rf ~/Library/Application\ Support/Leaf/cache # Linux rm -rf ~/.config/leaf/cache

Collaboration not syncing

Solution: Check your network connection and firewall settings. Ensure that ports 443 (HTTPS) and 8080 (WebSocket) are open. Restart the application if necessary.

GPU acceleration not working

Solution: Update your graphics drivers. Go to Settings → Graphics → GPU Rendering and toggle it on. If still not working, check if your GPU is supported:

  • NVIDIA: CUDA Compute Capability 3.0+
  • AMD: RDNA or RDNA 2 architecture
  • Intel: Arc A380 or newer

File export fails

Solution: Ensure you have write permissions to the export directory. Try exporting to your Documents or Desktop folder first. Check disk space availability (requires at least 500MB for video export).

High memory usage

Solution: Close unused projects, reduce canvas size, or lower preview quality in Settings. If working with large files, consider splitting your project into smaller artboards.
